i show that you can have more than two color stops in a gradient.
This page provides a demonstration of using gradients in your connectors. Additionally it sets the 'curviness' of the Bezier curves to 80.

Note: this does not work in IE. In IE you will see fallback values for stroking the connectors - a line with the same colour as the target of the connection.

This demonstration uses version 1.0.0 of jsPlumb.
