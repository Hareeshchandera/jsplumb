This is a demonstration of anchors that change position dependent on the location of the other element in a connection. Also, each connection is decorated with a jsPlumb.Overlays.Diamond, located halfway along the connector.

This demonstration uses jsPlumb 1.2.3, jQuery 1.4.2 and jQuery UI 1.8.0.
