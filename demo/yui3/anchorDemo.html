This page provides a demonstration of how to work with Anchors.

All the green windows are anchored to Window 1 using custom anchor positions on Window 1, except Window 4, which uses the BottomCenter anchor provided by jsPlumb.

Additionally, this page shows the usage of a different endpoint for each end of the connector, and a gradient in the connector. If you are using IE, you will see green connectors with no gradient. ExplorerCanvas, which jsPlumb uses for IE, does not support gradients.

This demonstration uses jsPlumb 1.2.6-RC1 and YUI 3.3.0.
