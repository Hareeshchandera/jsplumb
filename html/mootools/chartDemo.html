On this page we set our connectors to gray and specify an endpoint of radius 9.

The Bezier curve used on this page has a 'curviness' of 50; the Bezier connectors are a lot flatter than those on the first demo, which uses the default values. See the documentation for examples of how to set the Bezier curviness.

The elements of the chart are not draggable.

This demonstration uses jsPlumb 1.2, MooTools 1.2.4, and Drag.Move from MooTools More 1.2.2.4.
