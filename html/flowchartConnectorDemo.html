This page provides a demonstration of how to plug a new Connector type in to jsPlumb - in this example the code is included in the file 'jquery.jsPlumb-flowchart-RC1.js', which provides a connector style reminiscent of flowcharts.

The flowchart connector script is not yet finished - it currently does not support every different combination of endpoint locations, it needs refactoring, and also maybe some work on the transitions - but I thought it'd be good to put the demo up for people to play with.

This demonstration uses version 1.0.4-RC1 of jsPlumb and RC1 of the flowchart connectors.
